Message Blocks

How To Use The Blocks

When message received

This block runs the code inside of it when a message is received in any channel the bot can access to.

Example Of Usage

Be careful when using this, as the bot can respond to itself and it will start sending messages rapidly!

When message received & author is a user

This block is the same as when message received but if the author is a bot, the code will not run.

Example Of Usage

When you send a message in any channel the bot can access, your bot will respond with the message!
